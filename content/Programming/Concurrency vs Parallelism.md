# Concurrency vs Parallelism

Concurrency
- Context switching
- Can be done even with single core CPU
- Example:
	- Web server

Parallelism:
- Do multiple task in the same time
- Leverage multiple core of CPU
- Example:
	- Training Machine Learning
	- Rendering video
